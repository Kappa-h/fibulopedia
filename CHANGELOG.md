# Changelog

All notable changes to Fibulopedia will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-23

### Added

- Initial release of Fibulopedia
- Home hub page with clickable navigation cards
- Weapons page with sortable table and filtering
- Equipment page organized by slots
- Spells page with vocation filtering and search
- Monsters page with HP, EXP, and loot information
- Quests page with basic quest listings
- Map page with static map display
- Server Info page with rates and general information
- Advanced global search across all content types
- About page with credits and project information
- Dark fantasy theme with custom CSS styling
- Modular architecture with services layer
- Type-hinted code with comprehensive docstrings
- Logging infrastructure
- Unit tests for core services
- Example data for all content types

### Technical

- Built with Streamlit for rapid web development
- File-based data storage using JSON
- Fully typed Python codebase
- Separation of concerns: models, services, UI components
- Prepared for future multi-language support
- Mobile-friendly responsive design
